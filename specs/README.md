# Specs Index

This directory contains living specifications for the `goviesdeze` service.

- `specs/01-product-spec.md`: product goals, scope, and non-goals.
- `specs/02-system-design.md`: architecture, components, and request flows.
- `specs/03-api-spec.md`: HTTP endpoint contracts and behavior.
- `specs/04-data-and-storage-spec.md`: storage layout, usage accounting, and file matching rules.
- `specs/05-security-ops-spec.md`: security model, runtime behavior, and operational concerns.
- `specs/06-testing-and-quality-spec.md`: test strategy, quality gates, and current coverage.
- `specs/07-hexagonal-roadmap.md`: step-by-step refactor plan toward stronger hexagonal boundaries.

Status: these specs describe the code as of current `main`, including the recent extraction of `filequery` and `archivequery` core packages.
